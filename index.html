Q&A

1. שלושת ההבדלים העיקריים בין Web1, Web2 ו- Web3 הינם:

א. מבנה האינטרנט: בתחילת הדרך Web 1.0 היה בנוי על ידי מספר מצומצם של יוצרי תוכן וכמות גדולה מאוד של משתמשים שצרכו מידע זה. לעומת זאת Web 2.0 מדגיש יצירת תוכן על ידי משתמשים והשתתפות המשתמשים במבנה האינטרנט (דבר שהופך אותו ליותר חברתי בעצם). Web 3.0 אף מדבר על יצירת אפליקציות על ידי משתמשים וחברות יוצרות פלטפורמות שמאפשרות שיתוף בין אנשים

ב. מבנה דפי האינטרנט: ב- Web 1.0 ישנם בעיקר דפים סטאטיים, בעוד שב- Web 2.0 דפי האינטרנט מכילים תוכן דינאמי (בין היתר באמצעות AJAX ו- Javascript) והם ריספונסיביים למשתמש. ב- Web 3.0 המטרה היא אפילו לספק למשתמש חווית גלישה אישית.

ג. חיפוש באינטרנט: ב- Web 1.0 מנועי חיפוש הינם מהירים אך פעמים רבות אינם מדויקים. לעומת זאת, ב- Web 2.0 מנועי החיפוש מסתמכים בנוסף על תגיות (Tags) של מידע וכך מנועי החיפוש נהיית מדויקים יותר. ב- Web 3.0 פריטי המידע כבר מתויגים אוטומטית ולכן החיפוש נעשה מדויק אף יותר. בנוסף ב- Web 3.0 ישנה הבנה סמנטית של הפריטים באינטרנט וכך החיפוש מבוצע גם מבחינה סמנטית.


2. נדגים את ההבדלים על ידי חיפוש השאילתה "תופעות לוואי של תרופות":

Web 1.0- על המשתמש יהיה ממש לכתוב את ה- URL של אתר בו קיים המידע הזה, או שיוכל לחפש את השאילתה במנוע חיפוש, אך המידע שיאוחזר לא יהיה מאוד מדויק.

Web 2.0- המשתמש יכול לחפש את השאילתה במנוע החיפוש והוא יקבל עמודים בהם קיימות התגיות הקשורות לנושא.

Web 3.0- המשתמש יכול לחפש את השאילתה במנוע החיפוש והוא יקבל עמודים בהם קיימים פריטי מידע בעלי תגיות הקשורות לנושא, כאשר הרבה מתגיות אלה נוצרו אוטומטית. בנוסף המידע שיתקבל יהיה מדויק יותר ויתקבל לאחר ניתוח סמנטי של השאילתה ודפי האינטרנט.
